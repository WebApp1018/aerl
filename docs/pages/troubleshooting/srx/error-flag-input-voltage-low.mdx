# Error Flag: Low Input Voltage

| | |
|---|---|
| Fault code | `-1153` |
| LED code | None. |

Please note that this flag is expected to appear during low light conditions such as sunrise and sunset.

## Possible Causes

- Damaged or corroded PV array connection.
- Loose PV input terminals.
