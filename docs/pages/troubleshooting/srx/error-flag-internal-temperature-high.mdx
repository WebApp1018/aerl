# Error Flag: Internal Temperature Critical

| | |
|---|---|
| Fault code | `-1100` |
| LED code | Solid red. |

The device’s internal systems have reached a critical temperature and shutdown to prevent system damage.

## Possible Causes

- Unsatisfactory environmental conditions.
- Mounting or near an external wall that receives heating from the sun.
- Failure of the temperature sensor.

## Recommended Action

No action. The device will automatically restart when it’s safe to do so.
