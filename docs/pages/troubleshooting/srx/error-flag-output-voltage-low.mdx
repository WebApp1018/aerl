# Error Flag: Low Output Voltage

| | |
|---|---|
| Fault code | `-1159` |
| LED code | 2 flashes red. |

No output load has been detected.

## Possible Causes

- Damaged or corroded battery connections.
- Loose output terminals.
- Misconfigured battery voltage.
- Battery shutdown caused by battery management system.

## Recommended Action

Confirm the battery is connected to the controller.
