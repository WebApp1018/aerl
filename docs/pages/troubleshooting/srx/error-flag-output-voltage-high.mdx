# Error Flag: High Output Voltage

| | |
|---|---|
| Fault code | `-1154` |
| LED code | 3 flashes red. |

A high output voltage has been detected.

## Possible Causes

- Misconfigured battery voltage.
- Battery is being charged from a secondary source.

## Recommended Action

Confirm all system battery charge parameters are correct.

For systems with multiple chargers: offset the maximum charge voltage to stagger each chargers cutoff point.
