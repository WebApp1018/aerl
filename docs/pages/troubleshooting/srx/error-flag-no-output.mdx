# Error Flag: No Output Detected

| | |
|---|---|
| Fault code | `-350` |
| LED code | None. |

A low output voltage has been detected.

## Possible Causes

- Damaged or corroded battery connections.
- Loose output terminals.
- Battery shutdown caused by battery management system.
- Open battery protection breaker.

## Recommended Action

Confirm the battery is connected to the controller. Power cycle the device and contact AERL if the fault reoccurs.
